# CURA Healthcare Selenium Framework

End-to-end UI automation framework for the CURA Healthcare demo application.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager
- Page Object Model (POM)
- Log4j (planned)

## Automated Scenarios
✔ Login  
✔ Book appointment  
✔ Appointment confirmation validation (in progress)

## Framework Structure
```text
src/test/java
base
pages
tests
utils
```

## Features
- Explicit waits
- Assertions
- ChromeOptions handling
- Stable browser setup
